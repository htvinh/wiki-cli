The Quality Assurance department reports to the Plant Director. Each pro-
duction line has a designated QA Supervisor who is the first point of contact
for quality-related issues. The QA Manager oversees the entire QA department
| and chairs     | the Quality       | Management |               | Review     | meeting.   |                  |
| -------------- | ----------------- | ---------- | ------------- | ---------- | ---------- | ---------------- |
| Management     | Review            |            |               |            |            |                  |
| Top management | reviews           | the        | QMS           | quarterly. | The review | agenda includes: |
| • Status       | of corrective     | and        | preventive    | actions    |            |                  |
| • Audit        | results (internal |            | and external) |            |            |                  |
| • Customer     | feedback          | and        | complaints    |            |            |                  |
| • Process      | performance       | and        | product       | conformity |            |                  |
| • Status       | of preventive     | actions    |               |            |            |                  |
| • Follow-up    | actions           | from       | previous      | management | reviews    |                  |
| • Changes      | affecting         | the QMS    |               |            |            |                  |
| Continuous     | Improvement       |            |               |            |            |                  |
Acme Corporation uses the Plan-Do-Check-Act (PDCA) cycle for continuous
improvement. Allemployeesareencouragedtosubmitimprovementsuggestions.
The QA department tracks improvement initiatives through the CAPA system.
